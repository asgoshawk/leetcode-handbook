## 題目敘述

給定兩個非遞減排序陣列 `nums1` 與 `nums2`，其中 `nums1` 後方預留空間。請原地把兩者合併到 `nums1` 中。

```text
輸入：nums1 = [1,2,3,0,0,0], m = 3, nums2 = [2,5,6], n = 3
輸出：[1,2,2,3,5,6]
```

### 限制

- `nums1.length == m + n`。

## 核心觀念

`nums1` 後方有足夠空間。從兩個有效區段尾端比較，把較大值放到最後可寫入位置，能避免正向合併時覆蓋資料。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。

## 複雜度

- 時間：$O(m+n)$。
- 額外空間：$O(1)$。
