## 題目敘述

給定 linked list 的頭節點與整數 `val`，刪除所有值等於 `val` 的節點，回傳新的頭節點。

```text
輸入：head = [1,2,6,3,4,5,6], val = 6
輸出：[1,2,3,4,5]
```

## 核心觀念

刪除節點時最容易漏掉頭節點。使用 dummy 指向原 head，讓每個待刪節點都能由前一個節點重新接線。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。
