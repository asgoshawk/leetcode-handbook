## 題目敘述

若矩陣中某個元素為 0，請將該元素所在的整列與整欄都設為 0，要求原地修改。

```text
輸入：matrix = [[1,1,1],[1,0,1],[1,1,1]]
輸出：[[1,0,1],[0,0,0],[1,0,1]]
```

### 限制

- `1 <= m, n <= 200`。

## 核心觀念

先記錄第一列與第一欄本身是否需要清零，接著用它們標記其他列欄。第二輪依標記清零，最後處理第一列與第一欄。

### 不變量與正確性

演算法在每一步都維護上述狀態定義：已處理區段的答案不會被後續操作破壞，尚未處理區段則保留必要資訊。每次轉移都只根據目前元素與既有狀態更新，因此處理完所有輸入後，狀態即涵蓋所有可能答案。

## 複雜度

- 時間：$O(mn)$。
- 額外空間：$O(1)$。
