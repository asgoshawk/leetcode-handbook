## 題目敘述

給定長度為 `n` 的整數陣列 `nums`，其中保證存在一個出現次數嚴格大於 `floor(n / 2)` 的 majority element。請回傳該元素。

```text
輸入：nums = [3,2,3]
輸出：3

輸入：nums = [2,2,1,1,1,2,2]
輸出：2
```

### 限制

- `1 <= nums.length <= 50000`
- `-1000000000 <= nums[i] <= 1000000000`
- 輸入保證存在 majority element

## 核心觀念

維護候選值與票數。相同值加一，不同值減一；票數歸零時改用目前值。這等價於反覆刪除一對不同元素。

### 不變量與正確性

從陣列刪除任意一對不同值，不會改變 majority element：它至多被刪一次，而其他元素總數也少一次。持續配對消去後，保證占過半的值不可能全部消失，最後候選值必是答案。

## 複雜度

- 時間：$O(n)$。
- 額外空間：$O(1)$。
