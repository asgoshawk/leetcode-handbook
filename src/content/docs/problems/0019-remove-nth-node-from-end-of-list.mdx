## 題目敘述

給定單向鏈結串列 `head` 與整數 `n`，移除倒數第 `n` 個節點並回傳新的頭節點。

```text
輸入：head = [1,2,3,4,5], n = 2
輸出：[1,2,3,5]
```

### 限制

- 串列節點數至少為 1。
- `1 <= n <= 串列長度`。

## 核心觀念

倒數第 `n` 個節點等於從前面數第 `len - n` 個節點。先取得長度，再走到前一個節點即可刪除；快慢指標版本則用距離差維持同樣關係。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。

## 複雜度

- 時間：$O(n)$。
- 額外空間：$O(1)$；Rust 版本為了簡化所有權測試重建串列，使用 $O(n)$ 暫存。
