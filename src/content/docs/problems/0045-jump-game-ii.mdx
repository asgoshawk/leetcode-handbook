## 題目敘述

給定陣列 `nums`，每個位置的值代表最多可往後跳幾步。假設一定可以到達最後一格，回傳到達最後一格所需的最少跳躍次數。

```text
輸入：nums = [2,3,1,1,4]
輸出：2
```

## 核心觀念

目前一次跳躍能涵蓋的區間是 `[0, end]`。掃過這個區間時持續更新下一層能到的最遠位置 `far`，當走到 `end` 就代表需要再跳一次並切到下一層。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。
