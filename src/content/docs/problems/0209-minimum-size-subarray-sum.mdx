## 題目敘述

給定正整數陣列 `nums` 與整數 `target`，找出總和至少為 `target` 的最短連續子陣列長度；若不存在則回傳 0。

```text
輸入：target = 7, nums = [2,3,1,2,4,3]
輸出：2，子陣列 [4,3]
```

### 限制

- `1 <= nums.length <= 100000`
- `1 <= nums[i]`。

## 核心觀念

因為所有數都是正數，右端擴張只會讓總和增加，左端收縮只會讓總和減少。當總和達標時盡量收縮以取得最短長度。
