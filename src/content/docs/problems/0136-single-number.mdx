## 題目敘述

給定非空整數陣列 `nums`，其中恰好有一個元素只出現一次，其餘每個元素都剛好出現兩次。請找出只出現一次的元素。演算法必須是線性時間，且只能使用常數額外空間。

```text
輸入：nums = [2,2,1]
輸出：1

輸入：nums = [4,1,2,1,2]
輸出：4
```

### 限制

- `1 <= nums.length <= 30000`
- `-30000 <= nums[i] <= 30000`
- 除一個值外，其餘值皆恰好出現兩次

## 核心觀念

XOR 具有交換律與結合律，且 `x XOR x = 0`、`x XOR 0 = x`。把所有值 XOR 起來，成對值會抵消，只留下單獨值。

### 不變量與正確性

因交換律與結合律，可將相同值重新分組。每一對都變成 0，而 0 不影響 XOR 結果，因此最後結果恰為唯一沒有配對的值。

## 複雜度

- 時間：$O(n)$。
- 額外空間：$O(1)$。
