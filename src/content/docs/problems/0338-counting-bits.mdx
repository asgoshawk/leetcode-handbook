## 題目敘述

給定非負整數 `n`，回傳長度為 `n + 1` 的陣列，其中第 `i` 個值為 `i` 的二進位表示中 1 的數量。

```text
輸入：n = 5
輸出：[0,1,1,2,1,2]
```

### 限制

- `0 <= n <= 100000`。

## 核心觀念

`i` 的二進位等於 `i >> 1` 左移後加上最低位，因此 `bits[i] = bits[i >> 1] + (i & 1)`。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。

## 複雜度

- 時間：$O(n)$。
- 額外空間：$O(1)$，不含輸出。
