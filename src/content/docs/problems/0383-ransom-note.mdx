## 題目敘述

給定 `ransomNote` 與 `magazine`，判斷是否能使用 `magazine` 中的字元組成 `ransomNote`。每個字元只能使用一次。

```text
輸入：ransomNote = "aa", magazine = "aab"
輸出：true
```

## 核心觀念

這是典型 counting 問題。先統計可用字元，再掃過需求字串；任何字元數量不足就回傳 false。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。
