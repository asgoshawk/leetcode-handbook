## 題目敘述

給定字串陣列 `strs`，把互為 anagram 的字串放入同一組。兩個字串若包含完全相同的字元及次數，只是排列順序不同，就屬於同組。各組及組內字串可用任意順序回傳，空字串也必須正確分組。

```text
輸入：strs = ["eat", "tea", "tan", "ate", "nat", "bat"]
輸出：[["eat", "tea", "ate"], ["tan", "nat"], ["bat"]]
```

### 限制

- `1 <= strs.length <= 10000`
- `0 <= strs[i].length <= 100`
- 每個字串只含小寫英文字母。

## 核心觀念

互為 anagram 的字串排序後必定相同。把排序結果當成雜湊表鍵，原字串加入該鍵所對應的群組即可。

### 正確性

兩字串排序後相等，等價於它們對每個字元具有相同次數，也就等價於兩者互為 anagram。因此演算法把且只把互為 anagram 的字串分在一起。

## 複雜度

- 時間：$O(nk \log k)$；`k` 是最長字串長度。
- 額外空間：$O(nk)$。
