## 題目敘述

給定 binary tree，回傳 zigzag level order traversal：第一層由左到右，下一層由右到左，之後交替進行。

```text
輸入：root = [3,9,20,null,null,15,7]
輸出：[[3],[20,9],[15,7]]
```

## 核心觀念

BFS 天然按層處理。每層收集值後，根據目前方向決定是否反轉，接著切換方向。
