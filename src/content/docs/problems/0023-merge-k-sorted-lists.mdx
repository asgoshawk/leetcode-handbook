## 題目敘述

給定陣列 `lists`，其中每個元素都是一條依遞增順序排列的單向鏈結串列頭節點。請把所有串列的既有節點合併為一條已排序串列並回傳頭節點；輸入陣列或個別串列都可能為空。

```text
輸入：lists = [[1,4,5],[1,3,4],[2,6]]
輸出：[1,1,2,3,4,4,5,6]

輸入：lists = []
輸出：[]
```

### 限制

- `0 <= lists.length <= 10000`
- 所有串列合計最多 `10000` 個節點
- `-10000 <= Node.val <= 10000`
- 每條串列皆依遞增順序排列

## 核心觀念

兩兩合併串列，再把結果兩兩合併，如同 merge sort 的合併樹。每個節點在每一層最多被處理一次，合併層數為對數級。

### 不變量與正確性

兩條排序串列的合併會產生包含兩者全部節點的排序串列。每一層把互斥的串列群組正確合併；依此歸納，最後一條串列包含所有輸入節點且保持排序。

## 複雜度

- 時間：$O(N \log k)$，其中 $N$ 為節點總數。
- 額外空間：$O(\log N)$ 遞迴呼叫堆疊，不計輸出節點。
