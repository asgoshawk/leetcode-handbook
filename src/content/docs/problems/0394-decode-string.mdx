## 題目敘述

給定編碼字串，格式為 `k[encoded_string]`，代表括號內字串重複 `k` 次。請解碼完整字串，可能有巢狀編碼。

```text
輸入：s = "3[a2[c]]"
輸出："accaccacc"
```

### 限制

- 輸入保證格式有效。

## 核心觀念

遇到數字累積重複次數，遇到 `[` 就把目前字串與次數入 stack 並重置，遇到 `]` 時彈出並展開。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。

## 複雜度

- 時間：$O(L)$，`L` 為輸出長度。
- 額外空間：$O(L)$。
