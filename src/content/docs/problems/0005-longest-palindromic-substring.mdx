## 題目敘述

給定字串 `s`，回傳其中最長的 palindromic substring。若有多個答案，回傳任一個即可。

```text
輸入：s = "babad"
輸出："bab" 或 "aba"

輸入：s = "cbbd"
輸出："bb"
```

### 限制

- `1 <= s.length <= 1000`。

## 核心觀念

回文可能有奇數或偶數長度。對每個位置嘗試兩種中心，向外擴張到左右字元不同為止。

### 不變量與正確性

演算法在每一步都維護上述狀態定義：已處理區段的答案不會被後續操作破壞，尚未處理區段則保留必要資訊。每次轉移都只根據目前元素與既有狀態更新，因此處理完所有輸入後，狀態即涵蓋所有可能答案。

## 複雜度

- 時間：$O(n^2)$。
- 額外空間：$O(1)$，不含輸出。
