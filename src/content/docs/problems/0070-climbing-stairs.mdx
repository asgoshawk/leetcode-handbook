## 題目敘述

你正要爬上共有 `n` 階的樓梯，每次只能向上走 `1` 階或 `2` 階。請回傳抵達第 `n` 階共有多少種不同的走法；兩種走法只要任一步的步幅不同，就視為不同。

```text
輸入：n = 2
輸出：2
說明：可以走 1+1 或一次走 2 階。

輸入：n = 3
輸出：3
```

### 限制

- `1 <= n <= 45`

## 核心觀念

抵達第 `i` 階的最後一步必定來自 `i-1` 或 `i-2`，所以走法數為兩者相加。只保留前兩個狀態即可。

### 不變量與正確性

依最後一步分類的兩組走法互斥且涵蓋全部可能：走 1 階者與抵達 `i-1` 的走法一一對應，走 2 階者與抵達 `i-2` 的走法一一對應。因此遞迴式正確。

## 複雜度

- 時間：$O(n)$。
- 額外空間：$O(1)$。
