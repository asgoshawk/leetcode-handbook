## 題目敘述

給定元素互不相同的遞增陣列，它可能先在未知索引處旋轉，例如 `[0,1,2,4,5,6,7]` 可能成為 `[4,5,6,7,0,1,2]`。另給定 `target`，請回傳它目前的索引；若不存在則回傳 `-1`。演算法必須具有 $O(\log n)$ 時間複雜度。

```text
輸入：nums = [4,5,6,7,0,1,2]，target = 0
輸出：4

輸入：nums = [4,5,6,7,0,1,2]，target = 3
輸出：-1
```

### 限制

- `1 <= nums.length <= 5000`
- `-10000 <= nums[i], target <= 10000`
- `nums` 的值互不相同
- 陣列可能旋轉，也可能保持原狀

## 核心觀念

任一搜尋區間的中點至少有一側仍為遞增。先辨認有序半側，再判斷目標是否落在其值域中，以排除另一半。

### 不變量與正確性

每輪辨認出的有序半側可用端點精確判斷目標是否可能位於其中。保留包含目標的半側，或在目標不屬於其值域時保留另一側，因此不變量「若目標存在，必在搜尋區間內」持續成立。

## 複雜度

- 時間：$O(log n)$。
- 額外空間：$O(1)$。
