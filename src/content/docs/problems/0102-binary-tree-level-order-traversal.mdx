## 題目敘述

給定二元樹根節點 `root`，請依照由上到下、每層由左到右的順序回傳節點值。回傳值是一個二維陣列，其中第 `i` 個子陣列包含深度為 `i` 的所有節點值；空樹回傳空陣列。

```text
輸入：root = [3,9,20,null,null,15,7]
輸出：[[3],[9,20],[15,7]]

輸入：root = []
輸出：[]
```

### 限制

- 樹的節點數介於 `0` 與 `2000`
- `-1000 <= Node.val <= 1000`

## 核心觀念

使用 FIFO 佇列保存待走訪節點。每輪先記住當下佇列長度，恰好取出這些節點便是一整層；它們的子節點則留給下一輪。

### 不變量與正確性

開始每輪時，佇列前段恰好包含同一深度的全部節點。取出它們並依左右順序加入下一層節點，因此下一輪仍滿足相同不變量，最終依層完整輸出。

## 複雜度

- 時間：$O(n)$。
- 額外空間：$O(w)$，其中 $w$ 為樹的最大寬度。
