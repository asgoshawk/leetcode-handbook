## 題目敘述

給定兩棵二元樹的根節點 `p` 與 `q`，請判斷它們是否完全相同。兩棵樹必須具有相同結構，且每一個對應位置的節點值都相等；兩者皆為空樹時也視為相同。

```text
輸入：p = [1,2,3]，q = [1,2,3]
輸出：true

輸入：p = [1,2]，q = [1,null,2]
輸出：false
```

### 限制

- 兩棵樹的節點數皆介於 `0` 與 `100`
- `-10000 <= Node.val <= 10000`

## 核心觀念

同步比較兩個位置：同為空代表該位置相同，只有一個為空或值不同則失敗；否則遞迴比較左右子樹。

### 不變量與正確性

以樹高歸納。空樹情況直接正確；非空時，兩棵樹相同當且僅當根值相同且左右子樹分別相同，正好對應演算法的三個條件。

## 複雜度

- 時間：$O(n)$。
- 額外空間：$O(h)$，其中 $h$ 為樹高。
