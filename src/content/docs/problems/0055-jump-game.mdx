## 題目敘述

給定非負整數陣列 `nums`，每個位置的值代表最多可往前跳幾步。判斷是否能從索引 0 抵達最後一個索引。

```text
輸入：nums = [2,3,1,1,4]
輸出：true

輸入：nums = [3,2,1,0,4]
輸出：false
```

### 限制

- `1 <= nums.length <= 10000`。

## 核心觀念

從左到右掃描，只要目前索引不超過 `farthest` 就可抵達，並用此位置更新更遠可達範圍。若某個索引超過 `farthest`，代表中間斷開。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。
