## 題目敘述

給定字串陣列 `strs`，找出所有字串共同擁有的最長前綴；若沒有共同前綴，回傳空字串。

```text
輸入：strs = ["flower","flow","flight"]
輸出："fl"

輸入：strs = ["dog","racecar","car"]
輸出：""
```

### 限制

- `1 <= strs.length <= 200`
- `0 <= strs[i].length <= 200`。

## 核心觀念

先假設第一個字串是共同前綴，再逐一檢查其他字串。只要目前字串不以候選前綴開頭，就移除候選前綴最後一個字元。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。

## 複雜度

- 時間：$O(S)$，`S` 是所有字元總數。
- 額外空間：$O(1)$，不含輸出。
