## 題目敘述

給定非遞減排序陣列 `nums` 與 `target`，回傳目標值出現的第一與最後位置；若不存在，回傳 `[-1,-1]`。

```text
輸入：nums = [5,7,7,8,8,10], target = 8
輸出：[3,4]
```

### 限制

- `0 <= nums.length <= 100000`。

## 核心觀念

第一次二分找第一個 `>= target` 的位置，第二次找第一個 `> target` 的位置。若左界不是目標值，代表不存在。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。

## 複雜度

- 時間：$O(\log n)$。
- 額外空間：$O(1)$。
