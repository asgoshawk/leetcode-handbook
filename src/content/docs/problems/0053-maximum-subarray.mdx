## 題目敘述

給定整數陣列 `nums`，請從中選擇至少包含一個元素的連續子陣列，使其元素總和最大，並回傳該最大總和。子陣列必須使用相鄰元素，不可跳著選。

```text
輸入：nums = [-2,1,-3,4,-1,2,1,-5,4]
輸出：6
說明：連續子陣列 [4,-1,2,1] 的總和為 6。

輸入：nums = [-1]
輸出：-1
```

### 限制

- `1 <= nums.length <= 100000`
- `-10000 <= nums[i] <= 10000`

## 核心觀念

對每個位置，只需決定「接續前一段」或「從目前元素重新開始」。`ending` 表示以目前位置結尾的最大總和，`best` 保存所有位置中最大的 `ending`。

### 不變量與正確性

所有以位置 `i` 結尾的連續子陣列，不是只含 `nums[i]`，就是某個以 `i-1` 結尾的子陣列再接上它；取兩者最大即得到正確狀態。對所有結尾取最大值就是全域最佳解。

## 複雜度

- 時間：$O(n)$。
- 額外空間：$O(1)$。
