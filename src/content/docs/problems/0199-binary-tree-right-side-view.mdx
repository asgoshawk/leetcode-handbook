## 題目敘述

給定 binary tree 的根節點，想像從右側觀察它，回傳每一層可見的節點值。

```text
輸入：root = [1,2,3,null,5,null,4]
輸出：[1,3,4]
```

## 核心觀念

層序走訪時，每層最右邊的節點會最後被處理。記錄每層最後一個節點值，即可得到 right side view。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。
