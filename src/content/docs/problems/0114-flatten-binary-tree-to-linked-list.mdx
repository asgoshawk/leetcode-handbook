## 題目敘述

給定 binary tree 的根節點，原地將它攤平成 linked list。攤平後每個節點的左子節點都應為 `null`，右子節點順序需符合 preorder traversal。

```text
輸入：root = [1,2,5,3,4,null,6]
輸出：[1,null,2,null,3,null,4,null,5,null,6]
```

## 核心觀念

對每個有左子樹的節點，找到左子樹最右側節點，將原右子樹接到它的右邊，再把左子樹移到右邊並清空左指標。
