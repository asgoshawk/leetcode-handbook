## 題目敘述

給定整數陣列 `nums` 與視窗大小 `k`。視窗最初涵蓋前 `k` 個元素，之後每次向右移動一格，直到抵達陣列末端。請回傳每個位置視窗內的最大值，結果共有 `nums.length - k + 1` 個元素。

```text
輸入：nums = [1,3,-1,-3,5,3,6,7]，k = 3
輸出：[3,3,5,5,6,7]

輸入：nums = [1]，k = 1
輸出：[1]
```

### 限制

- `1 <= nums.length <= 100000`
- `-10000 <= nums[i] <= 10000`
- `1 <= k <= nums.length`

## 核心觀念

單調佇列保存仍在視窗內、且數值嚴格遞減的索引。加入新值前移除尾端不大於它的候選，因為新值更晚離開且至少同樣大；隊首永遠是最大值。

### 不變量與正確性

被尾端移除的索引不可能再成為最大值，因新索引值更大或相等且存活更久。移除過期隊首後，佇列包含所有未被支配的候選並依值遞減，因此隊首就是當前視窗最大值。

## 複雜度

- 時間：$O(n)$。
- 額外空間：$O(k)$。
