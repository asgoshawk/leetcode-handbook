## 題目敘述

給定整數陣列 `nums` 與整數 `k`，回傳總和等於 `k` 的連續子陣列數量。陣列可包含負數，因此不能使用單純滑動視窗。

```text
輸入：nums = [1,1,1], k = 2
輸出：2
```

### 限制

- `1 <= nums.length <= 20000`。

## 核心觀念

若目前 prefix sum 是 `s`，先前出現過 `s-k` 幾次，就有幾個以目前位置結尾的子陣列總和為 `k`。
