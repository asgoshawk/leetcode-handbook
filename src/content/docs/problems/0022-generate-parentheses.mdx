## 題目敘述

給定括號對數 `n`，產生所有由 `n` 對括號組成且合法的字串。

```text
輸入：n = 3
輸出：["((()))","(()())","(())()","()(())","()()()"]
```

### 限制

- `1 <= n <= 8`。

## 核心觀念

任何前綴中右括號數量不能超過左括號。遞迴時只在還有左括號時放 `(`，只在右括號剩餘多於左括號時放 `)`。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。
