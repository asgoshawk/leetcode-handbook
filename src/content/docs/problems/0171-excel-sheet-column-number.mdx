## 題目敘述

給定 Excel 欄位標題字串 `columnTitle`，回傳它對應的欄位編號。`A` 對應 1、`B` 對應 2，依此類推，`Z` 對應 26，`AA` 對應 27。

```text
輸入：columnTitle = "ZY"
輸出：701
```

## 核心觀念

這題等價於把字串當成 26 進位數字，但每一位是 `A = 1` 到 `Z = 26`，不是從 0 開始。由左到右掃描時，每讀到一個字元，就把目前答案乘以 26，再加上該字元的值。

### 不變量與正確性

處理完前 `i` 個字元後，`answer` 等於這段前綴代表的欄位編號。下一個字元加入時，原本前綴整體左移一位 26 進位，因此乘以 26，再加上新字元的值。掃描完所有字元後即為完整欄位編號。

## 複雜度

- 時間：$O(n)$。
- 額外空間：$O(1)$。
