## 題目敘述

給定 binary search tree 的根節點與整數 `k`，回傳其中第 `k` 小的節點值。

```text
輸入：root = [3,1,4,null,2], k = 1
輸出：1
```

### 限制

- `1 <= k <= 節點數`。

## 核心觀念

Binary Search Tree 的左子樹小於根，右子樹大於根，因此中序走訪順序就是由小到大。走到第 `k` 個節點即為答案。
