## 題目敘述

給定字串 `s`，找出第一個不重複字元的 index；若不存在，回傳 `-1`。

```text
輸入：s = "leetcode"
輸出：0
```

## 核心觀念

第一輪統計每個字元出現次數，第二輪依原順序找出 count 為 1 的第一個位置。保留原順序是關鍵。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。
