## 題目敘述

給定陣列 `nums` 與值 `val`，請原地移除所有等於 `val` 的元素並回傳剩餘元素數量 `k`。前 `k` 個位置需保存剩餘元素，順序可任意。

```text
輸入：nums = [3,2,2,3], val = 3
輸出：2，前兩個值為 [2,2]
```

### 限制

- `0 <= nums.length <= 100`。

## 核心觀念

走訪每個元素，只有當元素不等於 `val` 時才寫到 `write` 位置並前進。前 `write` 個位置即為保留結果。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。

## 複雜度

- 時間：$O(n)$。
- 額外空間：$O(1)$。
