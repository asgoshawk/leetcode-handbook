## 題目敘述

給定一組閉區間 `intervals`，每個區間為 `[start, end]`。請合併所有彼此重疊或邊界相接的區間，回傳一組互不重疊且涵蓋原輸入全部範圍的區間。答案須按照起點由小到大排列。

```text
輸入：intervals = [[1,3],[2,6],[8,10],[15,18]]
輸出：[[1,6],[8,10],[15,18]]

輸入：intervals = [[1,4],[4,5]]
輸出：[[1,5]]
```

### 限制

- `1 <= intervals.length <= 10000`
- `intervals[i].length == 2`
- `0 <= start <= end <= 10000`

## 核心觀念

先依起點排序。接著目前區間若與答案最後一段重疊，就延伸其終點；否則開啟新的答案區間。

### 不變量與正確性

排序後，尚未處理區間的起點不會早於目前區間。若下一段起點不大於目前終點，兩者聯集仍是一段區間；若更大，後續區間也不可能再與目前段重疊，因此可安全結束目前段。

## 複雜度

- 時間：$O(n \log n)$。
- 額外空間：$O(n)$，包含答案與排序實作所需空間。
