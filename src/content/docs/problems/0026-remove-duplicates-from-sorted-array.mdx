## 題目敘述

給定已排序整數陣列 `nums`，請原地移除重複值，使每個不同數字只保留一次，並回傳唯一值的數量 `k`。前 `k` 個位置需依原本相對順序放入唯一值，其餘位置內容不重要。

```text
輸入：nums = [1,1,2]
輸出：2，nums 前兩個值為 [1,2]
```

### 限制

- `1 <= nums.length <= 30000`
- `-100 <= nums[i] <= 100`
- `nums` 已由小到大排序。

## 核心觀念

因為陣列已排序，相同數值必然相鄰。用 `write` 指向下一個可寫入的位置，走訪時只在遇到新值時覆寫。
