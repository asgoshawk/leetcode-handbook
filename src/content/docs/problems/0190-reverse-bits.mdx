## 題目敘述

給定 32-bit unsigned integer，回傳其 bit 順序反轉後的數值。

```text
輸入：n = 43261596
輸出：964176192
```

## 核心觀念

重複 32 次：答案左移一位，加入 `n` 的最低位，接著 `n` 右移一位。這正好把原本由低到高的 bit 依序放到答案由高到低的位置。

### 不變量與正確性
