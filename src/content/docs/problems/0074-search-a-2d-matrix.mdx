## 題目敘述

給定 m x n 矩陣，每列由小到大排序，且每列第一個數大於前一列最後一個數。判斷 `target` 是否存在於矩陣中。

```text
輸入：matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 3
輸出：true
```

## 核心觀念

矩陣的排序條件等價於攤平成一維後仍為遞增。binary search 的 index `mid` 可映射回 `matrix[mid / n][mid % n]`。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。
