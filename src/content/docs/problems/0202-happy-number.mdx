## 題目敘述

對正整數 `n`，反覆將它替換成各位數平方和。若最後變成 1，稱為 happy number；請判斷 `n` 是否為 happy number。

```text
輸入：n = 19
輸出：true
```

### 限制

- `1 <= n <= 2^31 - 1`。

## 核心觀念

反覆把數字替換成各位數平方和。若到達 1 即為 happy number；若某個值重複出現，之後會無限循環，不可能到達 1。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。

## 複雜度

- 時間：$O(\log n)$ 到進入固定小範圍後循環。
- 額外空間：$O(1)$ 或 $O(k)$，依循環偵測方式而定。
