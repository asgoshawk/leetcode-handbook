## 題目敘述

給定 1-indexed 且非遞減排序的整數陣列 `numbers`，找出兩個數使總和等於 `target`，回傳兩個 1-indexed 位置。每組輸入恰有一組答案。

```text
輸入：numbers = [2,7,11,15], target = 9
輸出：[1,2]
```

### 限制

- `2 <= numbers.length <= 30000`
- `numbers` 已排序。

## 核心觀念

若左右兩端和太小，左指標必須右移才可能變大；若太大，右指標必須左移才可能變小。這個單調性讓我們不用枚舉所有配對。

### 不變量與正確性

演算法在每一步都維護上述狀態定義：已處理區段的答案不會被後續操作破壞，尚未處理區段則保留必要資訊。每次轉移都只根據目前元素與既有狀態更新，因此處理完所有輸入後，狀態即涵蓋所有可能答案。

- 額外空間：$O(1)$。
