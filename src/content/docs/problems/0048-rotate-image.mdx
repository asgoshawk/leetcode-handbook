## 題目敘述

給定 n x n 矩陣，將影像順時針旋轉 90 度。必須原地修改矩陣。

```text
輸入：matrix = [[1,2,3],[4,5,6],[7,8,9]]
輸出：[[7,4,1],[8,5,2],[9,6,3]]
```

## 核心觀念

順時針旋轉 90 度可以拆成兩步：先沿主對角線轉置，再把每一列反轉。這兩步都能原地完成。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。

## 複雜度

- 時間：$O(n^2)$。
- 額外空間：$O(1)$。
