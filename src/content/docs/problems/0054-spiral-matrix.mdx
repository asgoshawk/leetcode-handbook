## 題目敘述

給定 `m x n` 矩陣，依順時針螺旋順序回傳所有元素。

```text
輸入：matrix = [[1,2,3],[4,5,6],[7,8,9]]
輸出：[1,2,3,6,9,8,7,4,5]
```

### 限制

- `1 <= m, n <= 10`。

## 核心觀念

每走完上、右、下、左其中一邊，就把對應邊界往內縮。每次走下方或左方前要確認邊界仍有效，避免單列或單欄重複輸出。

### 不變量與正確性

演算法在每一步都維護上述狀態定義：已處理區段的答案不會被後續操作破壞，尚未處理區段則保留必要資訊。每次轉移都只根據目前元素與既有狀態更新，因此處理完所有輸入後，狀態即涵蓋所有可能答案。

## 複雜度

- 時間：$O(mn)$。
- 額外空間：除輸出外 $O(1)$。
