## 題目敘述

給定未排序整數陣列，找出最長連續整數序列的長度，要求時間複雜度為 $O(n)$。

```text
輸入：nums = [100,4,200,1,3,2]
輸出：4，序列為 [1,2,3,4]
```

### 限制

- `0 <= nums.length <= 100000`。

## 核心觀念

把所有數放入集合。若 `num - 1` 不存在，`num` 才是某段連續序列的起點，從起點往後數即可；每個數只會被展開一次。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。

## 複雜度

- 時間：平均 $O(n)$。
- 額外空間：$O(n)$。
