## 題目敘述

整數以十進位數字陣列表示，最高位在前且不含前導零。請將此整數加一，回傳新的數字陣列。

```text
輸入：digits = [1,2,3]
輸出：[1,2,4]

輸入：digits = [9,9]
輸出：[1,0,0]
```

### 限制

- `1 <= digits.length <= 100`
- `0 <= digits[i] <= 9`。

## 核心觀念

從尾端開始加一。若某位小於 9，直接加一並結束；若是 9，設為 0 並繼續向前進位。若全部都是 9，最前方補 1。

### 不變量與正確性

演算法在每一步都維護上述狀態定義：已處理區段的答案不會被後續操作破壞，尚未處理區段則保留必要資訊。每次轉移都只根據目前元素與既有狀態更新，因此處理完所有輸入後，狀態即涵蓋所有可能答案。

## 複雜度

- 時間：$O(n)$。
- 額外空間：除輸出外為 $O(1)$。
