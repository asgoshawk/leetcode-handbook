## 題目敘述

Fibonacci 數列定義為 `F(0) = 0`、`F(1) = 1`，且 `F(n) = F(n - 1) + F(n - 2)`。給定 `n`，回傳 `F(n)`。

```text
輸入：n = 4
輸出：3
```

## 核心觀念

直接遞迴會重複計算。迭代時只需要保留前兩項，就能由小到大推到第 `n` 項。

### 不變量與正確性
