## 題目敘述

原本嚴格遞增的陣列被旋轉若干次，請在沒有重複值的情況下找出最小元素。

```text
輸入：nums = [3,4,5,1,2]
輸出：1
```

### 限制

- `1 <= nums.length <= 5000`
- 所有元素互不相同。

## 核心觀念

若 `nums[mid] > nums[right]`，最小值一定在 `mid` 右側；否則最小值在 `mid` 或其左側。這是旋轉排序陣列仍保有的單調訊號。
