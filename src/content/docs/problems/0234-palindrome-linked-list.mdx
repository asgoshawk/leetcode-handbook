## 題目敘述

給定 singly linked list 的頭節點，判斷節點值由前往後與由後往前是否相同。

```text
輸入：head = [1,2,2,1]
輸出：true
```

## 核心觀念

可先把值收集成陣列，再以雙指標比較兩端。若要達到 $O(1)$ 額外空間，則可反轉後半段 linked list 後比較。
