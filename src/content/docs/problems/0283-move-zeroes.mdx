## 題目敘述

給定整數陣列 `nums`，請原地將所有 0 移到陣列尾端，同時保持非零元素的相對順序。

```text
輸入：nums = [0,1,0,3,12]
輸出：[1,3,12,0,0]
```

### 限制

- `1 <= nums.length <= 10000`。

## 核心觀念

先用 `write` 收集所有非零元素，保留它們原本相對順序；完成後從 `write` 到尾端補 0。

### 不變量與正確性

演算法在每一步都維護上述狀態定義：已處理區段的答案不會被後續操作破壞，尚未處理區段則保留必要資訊。每次轉移都只根據目前元素與既有狀態更新，因此處理完所有輸入後，狀態即涵蓋所有可能答案。

## 複雜度

- 時間：$O(n)$。
- 額外空間：$O(1)$。
