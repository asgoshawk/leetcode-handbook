## 題目敘述

給定 linked list 的頭節點，將節點每兩個一組交換，回傳交換後的頭節點。不能只交換節點值，必須實際調整節點連結。

```text
輸入：head = [1,2,3,4]
輸出：[2,1,4,3]
```

## 核心觀念

使用 dummy 節點讓交換頭兩個節點與中間節點共用同一套流程。每輪固定 `prev -> a -> b -> next`，改成 `prev -> b -> a -> next`，再把 `prev` 移到 `a`。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。
