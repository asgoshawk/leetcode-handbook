## 題目敘述

給定二元樹根節點 `root`，請判斷它是否為有效的 binary search tree。每個節點左子樹的所有值都必須嚴格小於該節點，右子樹的所有值都必須嚴格大於該節點，且左右子樹本身也須符合相同規則；重複值不合法。

```text
輸入：root = [2,1,3]
輸出：true

輸入：root = [5,1,4,null,null,3,6]
輸出：false
說明：值 3 位於 5 的右子樹，卻小於 5。
```

### 限制

- 樹的節點數介於 `1` 與 `10000`
- 節點值位於 32-bit signed integer 範圍

## 核心觀念

只比較父子節點不足以捕捉較深層違規。走訪時為每個節點帶入由所有祖先共同決定的開放區間 `(lower, upper)`。

### 不變量與正確性

根節點允許任意值。若節點值在祖先界線內，其左子樹再加入該值作上界，右子樹加入該值作下界。這恰好編碼 BST 對所有祖先的限制；所有節點皆通過當且僅當整棵樹有效。

## 複雜度

- 時間：$O(n)$。
- 額外空間：$O(h)$，其中 $h$ 為樹高。
