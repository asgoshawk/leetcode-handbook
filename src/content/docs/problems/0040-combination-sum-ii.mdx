## 題目敘述

給定可能含重複數字的 `candidates` 與 `target`，找出所有總和等於 `target` 的不重複組合。每個位置的數字最多使用一次。

```text
輸入：candidates = [10,1,2,7,6,1,5], target = 8
輸出：[[1,1,6],[1,2,5],[1,7],[2,6]]
```

### 限制

- `1 <= candidates.length <= 100`。

## 核心觀念

每個數只能使用一次，因此遞迴到下一層時從 `i + 1` 開始。同一層若遇到與前一個相同的值就跳過，避免重複組合。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。
