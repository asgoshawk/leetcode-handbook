## 題目敘述

給定二元樹，判斷是否為高度平衡：每個節點的左右子樹高度差都不超過 1。

```text
輸入：root = [3,9,20,null,null,15,7]
輸出：true
```

### 限制

- 節點數介於 0 到 5000。

## 核心觀念

若左右子樹高度差超過 1，就回傳失敗標記；否則高度為兩側最大高度加一。自底向上可以避免重複計算高度。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。

## 複雜度

- 時間：$O(n)$。
- 額外空間：$O(h)$ 遞迴深度。
