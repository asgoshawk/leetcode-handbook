## 題目敘述

給定已排序 linked list 的頭節點，刪除所有重複元素，使每個值只出現一次，回傳處理後的 linked list。

```text
輸入：head = [1,1,2,3,3]
輸出：[1,2,3]
```

## 核心觀念

因為 list 已排序，重複值一定相鄰。走訪時比較目前節點與下一個節點；若值相同就跳過下一個節點，否則前進。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。

## 複雜度

- 時間：$O(n)$。
- 額外空間：$O(1)$。
