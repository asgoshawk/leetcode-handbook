## Problem summary

Given a string `s`, return the length of its longest contiguous substring that contains no repeated character.

## 核心觀念

用 `[left, right]` 表示目前沒有重複字元的視窗。雜湊表保存每個字元「上次出現位置的下一格」；再次遇到相同字元時，`left` 可直接向右跳過舊位置，不必逐字移動。

### 不變量與正確性

處理 `right` 後，`left` 是能讓視窗結尾固定在 `right` 且不重複的最小合法位置。`left` 只能向右移，因此已排除的重複字元不會重新進入視窗。每個合法結尾都會被檢查，最大視窗長度就是答案。

<Aside type="caution" title="左界不能倒退">
  某字元上次出現的位置可能已在視窗外，因此更新時必須取 `max(left, lastSeen)`。
</Aside>

## 複雜度

- 時間：平均 $O(n)$；每個字元處理一次。
- 額外空間：$O(k)$；`k` 是不同字元數量。
