## 題目敘述

給定字串 `s`。將所有英文字母轉為小寫並移除非英數字元後，若結果由左讀到右與由右讀到左完全相同，就稱為 palindrome。請判斷 `s` 是否符合此條件；空的正規化結果視為 palindrome。

```text
輸入：s = "A man, a plan, a canal: Panama"
輸出：true

輸入：s = "race a car"
輸出：false

輸入：s = " "
輸出：true
```

### 限制

- `1 <= s.length <= 200000`
- `s` 僅包含可列印 ASCII 字元

## 核心觀念

左右指標分別跳過非英數字元，再以不分大小寫方式比較有效字元。相同就同時往內，任一對不同即可立即回傳 false。

### 不變量與正確性

每輪比較的是正規化字串中尚未檢查的最左與最右字元。若不同，必非 palindrome；若相同，移除這一對後，原問題等價於比較剩餘內部字串。

## 複雜度

- 時間：$O(n)$。
- 額外空間：$O(1)$。
