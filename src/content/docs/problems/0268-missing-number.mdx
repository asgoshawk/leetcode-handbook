## 題目敘述

給定長度為 `n` 的陣列，內容是不重複且來自 `0..n` 的數字，但少了一個。請找出缺少的數字。

```text
輸入：nums = [3,0,1]
輸出：2
```

### 限制

- `1 <= nums.length <= 10000`
- 所有數字皆不重複。

## 核心觀念

範圍是 `0..n` 且只缺一個數，因此完整總和減去目前總和就是答案。若擔心溢位，也可用 XOR 成對抵消。

### 不變量與正確性

演算法在每一步都維護上述狀態定義：已處理區段的答案不會被後續操作破壞，尚未處理區段則保留必要資訊。每次轉移都只根據目前元素與既有狀態更新，因此處理完所有輸入後，狀態即涵蓋所有可能答案。

## 複雜度

- 時間：$O(n)$。
- 額外空間：$O(1)$。
