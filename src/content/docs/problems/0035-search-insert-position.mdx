## 題目敘述

給定已排序且不重複的整數陣列 `nums` 與 `target`，若 `target` 存在則回傳其索引；否則回傳它應插入的位置，使陣列仍保持排序。

```text
輸入：nums = [1,3,5,6], target = 5
輸出：2

輸入：nums = [1,3,5,6], target = 2
輸出：1
```

### 限制

- `1 <= nums.length <= 10000`
- `nums` 嚴格遞增。

## 核心觀念

`left` 永遠是可能答案的最左界，`right` 是不含端點的右界。當 `nums[mid] < target` 時答案只能在右側，否則保留 `mid`。

### 不變量與正確性

演算法在每一步都維護上述狀態定義：已處理區段的答案不會被後續操作破壞，尚未處理區段則保留必要資訊。每次轉移都只根據目前元素與既有狀態更新，因此處理完所有輸入後，狀態即涵蓋所有可能答案。
