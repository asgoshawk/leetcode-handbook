## 題目敘述

字母 `A` 到 `Z` 分別對應 `1` 到 `26`。給定只含數字的字串 `s`，回傳它可以被解碼成字母字串的方式數。

```text
輸入：s = "226"
輸出：3
```

## 核心觀念

`dp[i]` 表示前 `i` 個字元的解碼方式。若單一字元不是 `0`，可接上 `dp[i-1]`；若最後兩位介於 `10` 到 `26`，可接上 `dp[i-2]`。

### 不變量與正確性

每一步都維護演算法狀態的定義，並只根據目前元素與已保存資訊更新。當狀態轉移保留所有仍可能成為答案的候選，且排除的候選已可證明不可能更好或不合法時，走訪結束後留下的狀態就是正確答案。
