## 題目敘述

陣列只包含 0、1、2，分別代表三種顏色。請原地排序，使相同顏色相鄰且順序為 0、1、2。

```text
輸入：nums = [2,0,2,1,1,0]
輸出：[0,0,1,1,2,2]
```

### 限制

- `1 <= nums.length <= 300`。

## 核心觀念

`0..low` 是 0，`low..i` 是 1，`high..end` 是 2。掃描到 0 就與 `low` 交換，掃描到 2 就與 `high` 交換且先不前進。

### 不變量與正確性

演算法在每一步都維護上述狀態定義：已處理區段的答案不會被後續操作破壞，尚未處理區段則保留必要資訊。每次轉移都只根據目前元素與既有狀態更新，因此處理完所有輸入後，狀態即涵蓋所有可能答案。
