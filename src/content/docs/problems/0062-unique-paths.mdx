## 題目敘述

在 `m x n` 網格左上角出發，只能向右或向下移動，請計算到右下角共有多少條不同路徑。

```text
輸入：m = 3, n = 7
輸出：28
```

### 限制

- `1 <= m, n <= 100`。

## 核心觀念

機器人只能往右或往下，因此到達 `(r,c)` 的最後一步只可能來自 `(r-1,c)` 或 `(r,c-1)`。用一維 DP 保存目前列即可。

### 不變量與正確性

演算法在每一步都維護上述狀態定義：已處理區段的答案不會被後續操作破壞，尚未處理區段則保留必要資訊。每次轉移都只根據目前元素與既有狀態更新，因此處理完所有輸入後，狀態即涵蓋所有可能答案。
